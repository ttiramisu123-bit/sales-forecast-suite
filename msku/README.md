# EVIL ENERGY 销售预测工作台 v1

这是一个本机离线版 HTML 工具，目标是逐步替代 Excel 公式计算。当前第一版已经支持：

- 导入最少必填模板：`SKU、价格、Type、Status、历史月销量`
- 自动按历史销量最后一个月份的下个月开始预测 12 个月
- 内置默认 SI 指数，并在缺失 Type/类目时提示补充
- 生成三条结果：公式预测、人工修正后最终预测、历史预测参考
- 曲线支持：SI 曲线、历史销量曲线、公式预测曲线、最终预测曲线、历史预测曲线
- 粒度支持：整体、Type/类目、单个 SKU
- 人工修正支持：目标月环比、SKU 策略系数、活动/清仓加量、SKU 某月直接改量
- 保存/加载独立项目 JSON
- 导出结果 CSV
- 导出只读 HTML 报告，适合放到共享盘/网盘给别人查看

## 使用方式

直接打开：

`E:\codex\sales-forecast-console\index.html`

## 基础模板字段

CSV / Excel 第一行建议使用：

```csv
SKU,价格,Type,Status,2025/7/1,2025/8/1,2025/9/1,2025/10/1,2025/11/1,2025/12/1,2026/1/1,2026/2/1,2026/3/1,2026/4/1,2026/5/1,2026/6/1
```

字段说明：

- `SKU` 或 `MSKU`：产品编码，必填
- `价格` 或 `Price`：单价，必填
- `Type` / `类目` / `品类` / `品线`：类目维度，必填
- 历史月份列：列名必须能识别成 `YYYY-MM`，例如 `2026-06`

## 历史预测版本

历史预测版本只用于参考曲线，不参与最终预测。月份应与本次未来预测月份一致，例如：

```csv
SKU,2026/7/1,2026/8/1,2026/9/1,2026/10/1,2026/11/1,2026/12/1,2027/1/1,2027/2/1,2027/3/1,2027/4/1,2027/5/1,2027/6/1
EVKT04002-AMZ1,20,23,26,28,30,32,29,31,42,38,36,34
```

## 当前限制

当前已经内置本地 SheetJS，可读取 `.xls/.xlsx/.csv/.tsv`。导出结果仍以 CSV、项目 JSON、只读 HTML 报告为主。
